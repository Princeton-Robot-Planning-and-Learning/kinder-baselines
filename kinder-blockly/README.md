# kinder-blockly

Visual programming interface for KinDER robot skills using Google Blockly.

## Installation

```bash
uv venv --python 3.10
source .venv/bin/activate
uv pip install -e ".[develop]"
```

## Usage

```bash
python -m kinder_blockly
```

Then open http://127.0.0.1:5000 in your browser.
